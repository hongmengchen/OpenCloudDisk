# 开发过程

1. 确定**分组**情况

   > 组员、组长、组名以及项目名称

2. 导入 **sql** 脚本，并命名 **yunpan**

3. 明确**版本控制**工具，且通过 **GitHub** 提供远程协作支持

   > - 安装 steam++ 用于加速 GitHub 
   > - 注册并熟悉 GitHub 
   > - 安装并学习 Git 

4. 明确**技术栈**

   > - 框架：Java EE (Spring、Spring MVC、MyBatis Plus)
   > - 数据库：MySQL

5. 明确软件**版本**

   > - Tomcat：9.X
   > - JDK：1.8

6. 完成初步**协同**编程

   > 组内成员每人都可以熟练提交和更新代码

7. 进行项目**分工**

   > - 确保项目的正常进行
   > - 细化分工参见 “3-项目分工.md”

8. 进入**开发**

9. 手写**实习报告**
